Segment Recognition

Task Settings
Recreate all Jellyfin media segments from cached analysis data on next scheduled task run. Resets automatically after completion.
Number of season/album groups to process in parallel during chromaprint comparison (default: 2).
Providers
Compare audio fingerprints from the end of episodes to detect credits/outro segments.
Duration from the end of the episode to fingerprint for credits detection (default: 240).
When a detected outro/credits ends before the episode ends, mark the remaining portion as a Preview segment. Requires credits fingerprinting.
Segment Duration Limits
Movie credits are typically much longer than TV episode credits (default: 900).
Black Frame Settings
Frames are downscaled to this height before analysis. Lower = faster. Uses GPU-side scaling when hardware acceleration is available.
Clear all cached black frame data and re-analyze on the next task run. Useful after changing the analysis resolution. Resets automatically.
Analysis Regions
Percentage of the episode to scan from the start for intro detection (default: 25%).
Duration from the end of the episode to scan for outro detection (default: 240s).
Refinement
Snap segment boundaries to nearby silence gaps for more natural transitions.
Noise threshold in dB for silence detection (default: -50).
Minimum silence duration in seconds to be considered a gap (default: 0.33).
Inward search window (toward segment center) for silence snapping (default: 5.0).
Outward search window (away from segment center) for silence snapping (default: 2.0).
Snap segment boundaries to nearby video keyframes for clean cuts.
Maximum distance from boundary to search for keyframes (default: 3.0).
Chapter Name Patterns
Names are matched using word-boundary matching (not substring). For example, "Intro" will match "Intro" but not "Introduction to Characters".
Chromaprint Settings
Maximum duration to fingerprint (capped to the intro analysis region).
